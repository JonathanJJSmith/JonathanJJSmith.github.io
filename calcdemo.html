Basic Calculator Demo

This program is a basic calculator written in Tkinter that is meant to do basic equations like 1+1, 2-2, 3*3, etc. Whenever you put in numbers and math signs in the program, math equations are being made. Once you press the equal sign, you get the response to what you inputted.

command=lambda: buttonUpdate(#)) is used to update what is being displayed on the screen

The 5 in command=lambda: buttonUpdate(5)) is the number that is displayed when the button updates the entry. Lambda: is a more compact version of using a list as when something is updated with the press of a button, it tells the command that is being updated to update the entry that displays the numbers and signs.
